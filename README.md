# How To

## Local Development

Auth is managed through the new httpApi event type

```bash
npm run dev
```

## Deploy

This example is made to work with the Serverless Framework dashboard which includes advanced features like CI/CD, monitoring, metrics, etc.

```bash
npm run login
npm run deploy
```
